# skillm — Documentation Index

`skillm` is a CLI that manages AI-agent **skills**: it fetches them from sources into a
single central store and links them into the skill folders that agents (Claude, Codex) read.

## Project-level docs
- [CONTEXT.md](./CONTEXT.md) — ubiquitous language / glossary (read first)
- [PLAN.md](./PLAN.md) — full implementation plan for v0.1.0 (architecture, commands, build order)

## Constraints
- **Platforms:** Linux + macOS only (amd64/arm64). No Windows.
- **Language:** Go.
